Prefix function - Knuth-Morris-Pratt

Definition You are given a string  sss  of length  nnn . The prefix function for this string is defined as an array  π\piπ  of length  nnn , where  π[i]\pi[i]π[i]  is the length of the longest proper prefix of the substring  s[0…i]s[0 \dots i]s[0…i]  which is also a suffix of this substring.

π[i]=max⁡k=0…i{k:s[0:k]=s[i−k+1:i+1]}\pi[i] = \max_ {k = 0 \dots i} \{k : s[0:k] = s[i-k+1:i+1] \}π[i]=maxk=0…i​{k:s[0:k]=s[i−k+1:i+1]}

For example, prefix function of string "abcabcd" is  [0,0,0,1,2,3,0][0, 0, 0, 1, 2, 3, 0][0,0,0,1,2,3,0] , and prefix function of string "aabaaab" is  [0,1,0,1,2,2,3][0, 1, 0, 1, 2, 2, 3][0,1,0,1,2,2,3]  By definition π[0]=0\pi[0] = 0π[0]=0

Brute force algo to compute π\piπ is O(n3)O(n^3)O(n3)

# O(n^3)
def kmp(s):
    pi = [0]*len(s)
    for i in range(len(s)):
        for k in range(i):
            if s[:k] == s[i-k+1:i+1]:
                pi[i] = k
    return pi

First optimization

Observe π(i+1)\pi(i+1)π(i+1) is at most 1 larger than π(i)\pi(i)π(i). Thus when moving to the next position, the value of the prefix function can either increase by one, stay the same, or decrease by some amount.

Algo: In total the function can grow at most  nnn  steps (after all interations), and therefore also only can decrease a total of  nnn  steps. This means we only have to perform  O(n)O(n)O(n)  string comparisons, and reach the complexity  O(n2)O(n^2)O(n2) .

# O(n^2)
def kmp(s):
    pi = [0]*len(s)
    for i in range(1,len(s)): # on each iteration, pi[i] increases at most with 1
        for j in range(pi[i-1]+1,0,-1):
            if s[:j] == s[i-j+1:i+1]: # hit this line at most n times during all iterations from the two loops above
                pi[i] = j # here pi decreases or increases at most by 1
                break
    return pi

def KMP(s):
    pi = [0]*len(s)
    for i in range(1,len(s)):
        j = pi[i-1]
        while j > 0 and s[i] != s[j]:
            j = pi[j-1]
        if j == 0: 
            pi[i] += s[i] == s[0]

            pi[i] = j+1

Counting the number of occurrences of each prefix

Given a string  sss  of length  nnn . In the first variation of the problem we want to count the number of appearances of each prefix  s[0…i]s[0 \dots i]s[0…i]  in the same string. 

def solve(s):
    pi = KMP(s)
    res = [0]*(len(s)+1)
    for i in range(len(s)):
        res[pi[i]] += 1
    for i in range(len(s)-1,0,-1):
        res[pi[i-1]] += res[i]
    for i in range(len(s)+1):
        res[i] += 1
    return res[1:]

# O(n**2)
class Solution:
    def longestPalindrome(self, s: str) -> str:
        def palin(i,j):
            while i >= 0 and j < len(s) and s[i] == s[j]:
                i -= 1
                j += 1
            return s[i+1:j]
        res = ''
        for i in range(len(s)):
            odd = palin(i,i)
            even = palin(i,i+1)
            if len(res)<len(odd): res = odd
            if len(res)<len(even): res = even
        return res

# Manacher Algorithm. Find Longest Palindrome in O(n) time.
class Solution:
    def longestPalindrome(self, s: str) -> str:  
        T = '$#'+'#'.join(s)+'#&'
        P = [0]*len(T)
        C,R = 0,0
        for i in range(len(T)-1):
            if R > i: # P[i] = (R>i) and min(R-i,P[2*C-i]) slows down
                P[i] = min(R-i,P[2*C-i])
            while T[i+P[i]+1] == T[i-P[i]-1]:
                P[i] += 1
            if R < i+P[i]:
                C,R = i,i+P[i]
        l = max(P)
        i = P.index(l)
        # P[2:-2:2] returns the sizes of largest palindrom for each i being the center (only odd length palindromes!)
        #P[2:-2] returns sized of largest palindromes (odd and even)
        return s[(i-l)//2:(i+l)//2]

def Manacher(s):
    T = '$#'+'#'.join(s)+'#&'
    P = [0]*len(T)
    C,R = 0,0
    for i in range(len(T)-1):
        if R > i:

            P[i] += 1
        if R < i+P[i]:
            C,R = i,i+P[i]
    return P[2:-2]
